# laravel-integral

Laravel 积分

## 环境需求

- PHP >= 7.2.5

## Installation

```bash
composer require larva/laravel-integral -vv
```

## for Laravel

This service provider must be registered.

```php
// config/app.php

'providers' => [
    '...',
    Larva\Integral\IntegralServiceProvider::class,
];
```
